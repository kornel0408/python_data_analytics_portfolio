## Projects Included

### 1. E-Commerce Analysis

This project analyzes an e-commerce dataset consisting of multiple tables such as users, products, orders, order items, reviews, and events.

#### Key Analysis Performed:

* Data merging and preparation
* Revenue and sales analysis
* Product performance analysis
* Customer behavior analysis
* Time series analysis (monthly trends)
* City-wise performance analysis
* Funnel analysis (view → cart → purchase)

#### Key Insights:

* Top customers contribute a significant portion of total revenue
* Certain products have high views but low purchases, indicating conversion issues
* Sales show seasonal trends during specific months
* Some cities have higher cancellation rates
